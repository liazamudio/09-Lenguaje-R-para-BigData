Lenguaje R para Big Data

Desarrollado en 1993 por Robert Gentleman y Ross Ihaka, R es un lenguaje de programación inspirado en el lenguaje S. Además, es un ambiente de programación de libre acceso con código abierto para el análisis estadístico y gráfico.

El lenguaje R tiene soporte en una comunidad de desarrolladores, los cuales continuamente aportan mejoras a la herramienta y la impulsan a mejorarse constantemente y estar a la vanguardia.
